# Tibia stats

## Overview

Django project that provides tools and statistics for players of Tibia.

## Features

1. **Worlds informations.**
    - [x] Basic information.
    - [ ] Online time counter.
2. **Character information.**
    - [x] Basic information.
    - [x] World transfers.
    - [x] Name changes.
3. **Experience change list.**
    - [x] Mainland.
    - [x] Rookgaard.
4. **Charm change list.**
    - [ ] Mainland.
    - [ ] Rookgaard.
5. **Character bazaar statistics.**
    - [ ] Monitoring active auctions.
    - [ ] History of finished auctions.
    - [ ] Price statistics.
6. **Boss information.**
    - [ ] Boss list.
    - [ ] Boss statistics.
7. **Calculators.**
   - [ ] Training calculator.
   - [ ] Loot calculator.
   - [ ] Stamina calculator.
